# Soc Ops 🎯

> A fast, playful **social bingo** experience for in-person mixers, team offsites, and community events.

Meet someone new, match a prompt, and race to 5 in a row.

---

## Why Soc Ops?

- 🤝 **Break the ice quickly** with prompts that start real conversations
- ⚡ **Simple game loop** that works great for live events
- 🧠 **Agent-lab friendly** project for practicing design and multi-agent workflows
- 🐍 **Python + FastAPI** codebase that is easy to run and extend

## How it plays

1. Start a new board
2. Find people who match each square prompt
3. Mark squares as you go
4. Hit bingo with 5 in a row

---

## 🚀 Quick Start

For prerequisites and full setup, start with **[Part 00: Overview](https://copilot-dev-days.github.io/agent-lab-python/docs/step.html?step=00-overview)**.

If your environment is ready, run the app locally:

```bash
uvicorn app.main:app --reload
```

Then open `http://127.0.0.1:8000`.
